Key DAX Measures & Business Logic

Core Headcount Metrics

TotalEmployees = DISTINCTCOUNT(DimEmployee[EmployeeID]) ActiveEmployees = CALCULATE( [TotalEmployees], DimEmployee[Attrition] = "No" ) InactiveEmployees = CALCULATE( [TotalEmployees], DimEmployee[Attrition] = "Yes" )

Time Intelligence for Cohorts

TotalEmployeesDate = CALCULATE( [TotalEmployees], USERELATIONSHIP( DimEmployee[HireDate], DimDate[Date] ) ) % Attrition Rate Date = DIVIDE( [InactiveEmployeesDate], [TotalEmployeesDate], 0 )
Technical Highlight: Multiple inactive relationships managed through USERELATIONSHIP() enable flexible analysis across satisfaction dimensions, rating levels, and time contexts
